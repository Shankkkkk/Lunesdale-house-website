# 🏡 Lunesdale House - Live Commercial Website
**Live Site:** [https://lunesdalehouse.co.uk](https://lunesdalehouse.co.uk)

## 📌 Project Overview
This repository contains the custom front-end architecture and interactive UI components I developed for **Lunesdale House Residential Care**. 

This project was built from the ground up during my placement year as an IT Analyst and Lead Developer, as part of my **BEng in Computer Science at Anglia Ruskin University**. The goal was to bypass rigid, standard WordPress templates and engineer a highly bespoke, premium digital experience.

## 🛠️ Tech Stack
* **Markup & Styling:** HTML5, Tailwind CSS (injected via custom scripts)
* **Interactivity:** Vanilla JavaScript
* **Backend Integration:** PHP (Custom form handling & lead generation)
* **CMS Environment:** WordPress & Elementor

## 🚀 Key Technical Achievements
* **Overriding Theme Constraints:** Successfully bypassed strict Astra theme container limitations using custom viewport width configurations and Elementor canvas overrides to achieve edge-to-edge, modern layouts.
* **Custom Concierge Modals:** Engineered sleek, slide-out JavaScript modals (`opacity` and `transform` transitions) for tour bookings and digital brochure requests, preventing users from having to navigate away from their current page.
* **Scroll-Reveal Animations:** Implemented Intersection Observer APIs to trigger smooth, high-end CSS animations as elements enter the viewport.
* **Tailwind Integration:** Injected a localised Tailwind CSS config directly into the WordPress header to allow for rapid, utility-first styling without bloating the core theme files.

## 📂 Repository Contents
Because this is a live WordPress environment, this repository does not contain core WP system files or sensitive backend credentials. Instead, it showcases the custom front-end components I engineered:
* `global-header.html` - Contains the Tailwind styling engine and responsive navigation.
* `global-footer.html` - Contains the PHP contact forms and JS modal triggers.
* `homepage.html` - The core landing page layout and scroll animations.
